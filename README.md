# Call of Duty: Warzone Hub

**Live Site:** 

This website provides:

- Dynamic player stats from Tracker.gg
- Latest Warzone news from the official blog
- RowdyRogan YouTube playlists embedded
- Download link for Call of Duty: Warzone

## Repository Structure

- `_assets/` → CSS, JS, images
- `_includes/` → Navbar HTML
- `_data/` → Playlist info JSON
- `index.html` → Main dashboard
- `about.html` → Optional instructions or About page
